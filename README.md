
# 🚀 SAP MM Automation & Engineering Hub

Este repositorio centraliza mis proyectos de arquitectura de datos, optimización de procesos en SAP S/4HANA (Módulo MM) y automatización de flujos de trabajo mediante n8n y herramientas de integración, así como la documentación estructurada para la preparación de la certificación oficial SAP MM **C_TS452**, orientada a escenarios reales y casos de negocio, desarrollada como material de apoyo para mis alumnos.

## 🛠️ Stack Tecnológico

- **ERP:** SAP S/4HANA (Módulo Materials Management).
    
- **Automatización:** n8n (Workflows vía API/IMAP).
    
- **Gestión de Conocimiento:** Obsidian (Arquitectura de Segundo Cerebro).
    
- **Control de Versiones:** Git / GitHub.
    

## 📊 Proyectos Destacados

### 1. Reingeniería del Proceso de Inventario (TS452)

Documentación y mapeo técnico del ciclo completo de inventario físico en SAP MM.

- **Alcance:** Desde la creación de documentos de inventario (`MI01`) hasta el análisis y contabilización de diferencias (`MI07`).
    
- **Entregable:** Mapa de procesos de alta resolución localizado en `/assets/`.
    

### 2. Inbox Zero Automation (Gmail + n8n)

Desarrollo de un middleware en n8n para la gestión inteligente de correos técnicos y alertas de procesos.

- **Lógica:** Filtrado dinámico por antigüedad (+3 días) y limpieza automática mediante protocolos de integración.
    
- **Objetivo:** Optimizar el tiempo de respuesta y reducir el ruido en comunicaciones.
    

## 📂 Estructura del Repositorio

- `📂 docs/`: Documentación técnica de tablas SAP (MARA, etc.) y guías de automatización.
    
- `📂 assets/`: Diagramas de arquitectura, mapas de procesos y recursos visuales.
